innerHTML / innerText
자바스크립트에서 특정 요소(Element)가 가지고 있는 값(Content)을 가지고 온다거나
또는, 값(Content)를 변경하려고 할 때 innerHTML, innerText 라는 속성을 이용

=> 가지고 올 때 : 요소객체.innerHTML / innerText
=> 변경할 때 : 요소객체.innerHTML / innerText = '변경할 값';
