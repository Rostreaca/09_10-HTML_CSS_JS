선택자(Selector)란? 중요 ☆★☆

특정 HTML요소를 선택하려고 할 때 사용하는 기능
선택자를 이용해서 원하는 요소만 쏙 선택해서 style속성을 부여

인라인 스타일 방식

1. 전체(모든 요소) 선택자

        [ 표현법 ]

        * {
            스타일속성 : 스타일속성 값;
            스타일속성 : 스타일속성 값;
        }

2. 태그 선택자

        [ 표현법 ]
        
        태그명 {
            스타일속성 : 스타일속성 값;
            스타일속성 : 스타일속성 값;
        }

3. 아이디 선택자(#아이디속성값)

        [ 표현법 ]

        #아이디속성값{
            스타일속성 : 스타일속성값;
            스타일속성 : 스타일속성값;
        }
